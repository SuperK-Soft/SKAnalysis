This is a Tool wrapper for the various different lfallfit functions. It chooses an appropriate version based on the SK geometry (phase), and whether the input is MC or data. For SK-\/\+IV, it currently uses {\ttfamily lfallfit\+\_\+sk4\+\_\+final\+\_\+qe43\+\_\+mc\+\_\+} for MC and {\ttfamily lfallfit\+\_\+sk4\+\_\+final\+\_\+qe43\+\_\+} for data. These are currently hard-\/coded, so comment out the one you want (or add a config variable to control it).\hypertarget{md_UserTools_lfallfit_README_autotoc_md34}{}\doxysection{Data}\label{md_UserTools_lfallfit_README_autotoc_md34}
Hits from the {\ttfamily skt\+\_\+}, {\ttfamily skq\+\_\+} and {\ttfamily skchnl\+\_\+} common block, which should be populated for any kind of SK file.

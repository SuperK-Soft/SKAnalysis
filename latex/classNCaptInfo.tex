\doxysubsection{Detailed Description}
This is an abstract base class for a Tool that converts neutron tagging information into a generalised form. Add support for your own ntagger by making a class that derives from this, and overrides the pure virtual functions Init\+Candidate\+Reader and Get\+Candidates. See existing cases for examples.
